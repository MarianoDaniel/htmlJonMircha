Etiquetas semánticas de estructura: delimitan el area de nuestra web

header- cabecera de un sitio web o de una sección
main- Define la sección principal del documento, solo puede existir una etiqueta main por documento
footer- pié de página de un sitio web o de una sección
nav- Represente una navegación...botonera etc
article - Representa una sección autocontenido (que por sí sola se explica), independiente del contexto
aside- Representa contenido complementario o secundario (blog articulos relacionados
section- Representa una sección de contenido genérico
address- Representa la información de contacto
☝️

Diferencia entre las etiquetas de bloque y las etiquetas de linea

La etiqueta de bloque por exelencia es div, una etiqueta de bloque es aquella que genera saltos de linea con sus etiquetas hermanas
la etiqueta de linea por exelencia es span, una etiqueta de linea solo ocupa el espacio necesario que tiene su contenido Hola, soy otra etiqueta span
